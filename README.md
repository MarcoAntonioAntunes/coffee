# PROJECT Coffee Week

## Requirements

For development, you will only need Node.js installed on your environment.

## Install

    $ cd coffee-week
    $ npm install

## Build, Start & Watch

    $ npm run start

## Run tests

    $ npm test

---

## Languages & tools

### JavaScript

- [React](http://facebook.github.io/react) is used for UI.

### CSS

- Vanilla CSS with Flexbox
